pop( )

El método pop() elimina y devuelve el último elemento de un array.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        var lastElement = vowels.pop();
        
        /*
        RESULTADO
        vowels ----> ['a', 'e', 'i', 'o']
        lastElement ----> 'u'
        */

shift( )

El método shift() elimina y devuelve el primer elemento de un array.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        var firstElement = vowels.shift();
        
        /*
        RESULTADO
        vowels ----> ['e', 'i', 'o', 'u']
        firstElement ----> 'a'
        */

reverse( )

El método reverse() invierte el orden de los elementos de un array.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        vowels.reverse();
        
        /*
        RESULTADO
        vowels ----> ['u', 'o', 'i', 'e', 'a']
        */

push( )

El método push() añade nuevos elementos en un array y devuelve la nueva longitud de este.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        var newLength = vowels.push('A', 'E');
        
        /*
        RESULTADO
        vowels ----> ['a', 'e', 'i', 'o', 'u', 'A', 'E']
        newLength ----> 7
        */

splice( )

El método splice() añade o elimina(según los parametros que le pasemos) elementos en un array, y devuelve los elementos eliminados.

Con el primer parámetro indicamos la posición en la que queremos realizar los cambios

Y con el segundo parametro, si le pasamos un 0 estamos indicando que vamos a insertar un elemento(o varios) en la posición indicada en el primer parametro. Mientras que si introducimos un número mayor que 0, eliminaremos el número de elementos que hayamos indicado o serán reemplazados en caso de que hayamos indicado nuevos elementos(más parametros). 


        //AÑADIENDO ELEMENTOS
        var vowels = ['a', 'o', 'u'];
        vowels.splice(1, 0, 'e', 'i');
        //RESULTADO vowels ----> ['a', 'e', 'i', 'o', 'u']

        //ELIMINANDO ELEMENTOS
        var vowels = ['a', 'e', 'i', 'o', 'u'];       
        var deletedElements = vowels.splice(2, 2);
        /*RESULTADO
        vowels ----> ['a', 'e', 'u']
        deletedElements ----> ['i', 'o']
        */
        

        //REEMPLAZANDO ELEMENTOS
        var vowels = ['a', 'w', 'i', 'o', 'u'];
        vowels.splice(1, 1, 'e');
        //RESULTADO vowels ----> ['a', 'e', 'i', 'o', 'u']

indexOf( )

El método indexOf() busca elementos en un array y devuelve su posición, en caso de no encontrarlo devolverá -1.

En caso de haber varios elementos con el mísmo nombre nos devolverá la posición del primero de ellos.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        var posElement = vowels.indexOf('i');
        
        /*
        RESULTADO
        vowels ----> ['a', 'e', 'i', 'o', 'u']
        posElement ----> 2
        */

fill( )

El método fill() reemplaza todos los elementos de un array con el parametro que le pasemos.

También es posible indicarle el valor seguido de la posición de inicio y final para que reemplace únicamente esos elementos.


        var vowels = ['a', 'e', 'i', 'o', 'u'];

        vowels.fill('A');
        
        /*
        RESULTADO
        vowels ----> ['A', 'A', 'A', 'A', 'A']
        */
